1. Go to [https://www.criminalip.io/](https://www.criminalip.io/) and create a free account.
2. Go to [https://www.criminalip.io/mypage/information](https://www.criminalip.io/mypage/information) after logging in.
3. Under **API Key**, copy your key.

!!! info
    CriminalIP currently supports IPv4 addresses only. IPv6 queries are not supported.

You can fill the `secrets.json` accordingly with the variable `"criminalip_api_key"` or the environment variable `CRIMINALIP_API_KEY` in your custom docker-compose file.
